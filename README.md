## Getting Started

1. install packages
```bash
npm install --force

cd backend

$backend> npm install --force
```

1. run the nextjs server:

```bash
npm run dev
```

2. run the backend server (express + mongodb)

```bash
cd backend

$backend> npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
